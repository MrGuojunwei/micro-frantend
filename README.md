# 基于qiankun框架实现微前端demo
> 该应用主要包含了一个主应用masterApp，两个子应用app1,app2，子应用与主应用的技术栈都是react，ui组件使用ant design组件库，打包工具使用webpack

克隆仓库：
git clone https://github.com/MrGuojunwei/micro-frontend.git

切换到项目目录
cd .\micro-frontend

安装依赖
npm install

启动
npm start
